# ⛏️ Minecraft, O Jogo

Uma página web simples e interativa criada para homenagear o universo de **Minecraft**. O projeto apresenta um breve histórico sobre a trajetória do jogo, destaca algumas das principais atualizações que marcaram a comunidade e conta com um sistema interativo de curtidas.

## 🚀 Funcionalidades

- **Histórico Informativo:** Detalhes sobre a criação do jogo por Markus "Notch" Persson, o desenvolvimento pela Mojang e a aquisição pela Microsoft.
- **Seção de Destaques:** Lista estilizada com algumas das atualizações mais memoráveis (Caves & Cliffs, Village & Pillage e The Redstone Update).
- **Interatividade:** Um botão de curtidas funcional que incrementa o contador em tempo real a cada clique.
- **Design Temático:** Cores personalizadas inspiradas no ambiente do jogo, com efeitos visuais e transições ao passar o mouse (*hover*) nos textos.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as tecnologias fundamentais da web:

- **HTML5:** Estruturação semântica do conteúdo da página.
- **CSS3:** Estilização visual, uso de fontes coloridas personalizadas por ID e alinhamento responsivo.
- **JavaScript (ES6):** Manipulação do DOM para o funcionamento da lógica do botão de curtidas.
